Проект "MINItable"
    Цель проекта: создать приложение под названием "MINItable" для быстрой работы с данными через графический интерфейс
Комплектация проекта:
    1.База данных, куда заносятся зарегстрировавшиеся пользователи users.db
    2.ui-файлы для классов: 'ui_files/main.ui'- Main(отобращение основного виджета), 'ui_files/startWindow' - StartWindow(
    отображение окна регистрации), 'ui_files/workwithdb' - DBSample(окно для работы с sql запросами)
    3.Главный класс, для осуществления взаимодействия пользователя с основным окном 'Main'
    4.Класс окна регистрации  и входа 'Start'
    5.Класс для проверки надежности пароля 'CheckPassword'
    6.Класс для генерации надежного пароля и логина 'Generation'
Задачи проекта:
    1.Написание логики работы основного окна для работы с данными
    2.Написание логики регистрации и входа пользователя
        2.1.Сохранения авторизовавшихся пользователей в бд
        2.2.Проверка пароля на надженость
        2.3.Проверка на уникальность логина
        2.4.Генерация надежного пароля и уникального логина
    3.Создание окна для работы с sql
    4.Осуществление вывода мер центральной тенденции данных по выбранной колонке
    5.Осуществление возможности заполнения пропусков
Необходимые библиотеки:
    1.PyQt5 Для работы с интерфейсом
    2.Pandas Для работы с данными
    3.sqlite3 Для работы с бд
    4.xlrd Для работы с xsl файлами
    5.csv Для работы с csv файлами
